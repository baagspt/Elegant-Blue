# Undangan Pernikahan Digital - Tema Art-07

Website undangan pernikahan digital modern dengan tema Art-07 yang elegan dan minimalis dengan dominasi warna hijau sage/olive muda dan aksen emas.

## Fitur Utama

- Desain responsif yang indah di semua perangkat
- Tema Art-07 dengan palet warna hijau sage, krem, dan aksen emas
- Kombinasi font serif dan sans-serif yang elegan
- Ornamen floral/hiasan tanaman halus
- Cover screen dengan salam perkenalan tamu
- Musik latar otomatis setelah tombol dibuka
- Bagian pengantin dengan foto, nama, dan kutipan
- Detail acara (akad & resepsi) dengan tombol Google Maps
- Countdown timer menuju tanggal pernikahan
- Bagian kisah cinta (Our Story) dengan timeline
- Galeri foto 5-8 gambar
- Protokol kesehatan
- Amplop digital dengan transfer bank
- Buku tamu dan formulir RSVP

## Teknologi yang Digunakan

- HTML5
- CSS3 (dengan Tailwind CSS untuk beberapa styling)
- JavaScript vanilla
- Google Fonts (Playfair Display & Montserrat)

## Penggunaan

1. Buka `index.html` di browser Anda
2. Anda dapat menambahkan nama tamu melalui parameter URL: `index.html?to=NamaTamu`
3. Sesuaikan data pengantin, tanggal, dan detail acara di file `index.html`
4. Ganti gambar placeholder di folder `images/` dengan foto asli
5. Ganti file musik di folder `audio/` dengan musik pilihan Anda

## Penyesuaian Data

Untuk menyesuaikan informasi undangan:

1. Buka `index.html`
2. Ganti nama pengantin di bagian cover dan couple section
3. Perbarui detail orang tua
4. Ubah tanggal dan waktu pernikahan
5. Perbarui lokasi acara
6. Ganti kutipan/ayat suci
7. Sesuaikan timeline our story
8. Ganti nomor rekening di bagian amplop digital
